ツールチップにリンク先のドメインを表示するテスト

HTML の title 属性に文字が入ってる要素にマウス乗っけるとブラウザデフォルトのツールチップが表示されますが、そこにリンク先のドメインを表示したいというわけです。
まあいちいちこんなことやらなくても、ブラウザの左下あたりにリンク先の URL は表示されるのですが、見ている場所を移動するのが面倒なのでドメイン名だけ表示してリンク先がどこなのか知らせることでさりげなく伝えることができるかもなーと思いました。

ただし、それは title 属性が入っていないリンクに限り、最初から指定されている場合にはそれを表示するようにします。
そのテストです。

除外テスト。www.yahoo.co.jp と www.bing.com は除外。
