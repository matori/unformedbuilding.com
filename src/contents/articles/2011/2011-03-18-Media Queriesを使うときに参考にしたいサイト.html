ライブラリ・プラグインなど

css3-mediaqueries.js
CSS3 Media Queries非対応ブラウザでもMedia Queriesを使えるようにするためのJavaScriptライブラリ。他のライブラリには依存していない。
Respond.js
同じく非対応ブラウザでもMedia Queriesを使えるようにするためのJavaScriptライブラリ。他のライブラリには依存していない。
320 and up
これも単体で動く。Media Queriesを使えるようにする。機能をサポートする他のスクリプトを同梱している。機能が多いので少し容量が大きい。
Adapt.js - Adaptive CSS
正確にはちょっと違うけど、似たような機能を提供するためのJavaScriptライブラリ。単独で動く。link要素で指定したCSSを切り替えるタイプ。960 Grid Systemの人が作ったみたい。
Media Queries / Dynamic resolution dependent layouts
同じく非対応ブラウザでもMedia Queriesを使えるようにするためのスクリプト。こちらはjQueryプラグイン。
Window Size bookmarklet to test/debug CSS3 Media Queries
JossCrowcroft.comによるブックマークレット。現在のウィンドウサイズを表示してくれる。
