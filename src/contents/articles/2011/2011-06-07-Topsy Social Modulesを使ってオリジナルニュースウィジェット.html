TopsyのSocial Modulesに何か他の使い道はないものかと思ってやってみました。

Social Modulesのカスタマイズに関しては、以前書いた「Topsyウィジェットの設定について」をご確認ください。

このウィジェットではTopsyの検索コマンドがそのまま使えるので、それを利用してニュース配信みたいなことができるんじゃないかなーと思いました。
気になる記事を見かけたらTwitterにポストして、それを拾ったTopsyからリンクを引っ張ってくるという感じです。

サンプル

（この段落に実際に動作するウィジェットがありましたが、Topsyは2015年12月16日にサービスを終了したため、ウィジェットを削除しました）

これのコードは以下のようになっています。

<div id="topsy_search_widget">
<script type="text/javascript">
var topsy_settings = {
  width            : '620',
  height           : '480',
  title            : 'まとりにゅーす',
  query            : 'from:ub_pnr',
  display_query    : '',
  data_type        : 'links',
  shell_color      : '#101010',
  shell_bg_color   : '#eaeaea',
  results_color    : '#303030',
  results_bg_color : '#fff',
  link_color       : '#08c',
  show_nick        : true,
  nick             : '@ub_pnr',
  allow_lang       : 'any'
}
</script>
<script type="text/javascript" src="http://cdn.topsy.com/search-widget/2.0/widget-loader.js"></script>
</div>

data_typeを'tweets'に変更すれば、そのときのツイートをそのまま表示できます。
記事にちょっとコメントつけてツイートするなら、こっちのほうがいいかもしれません。
また、自分のブログへのリンクを載せたくない場合は、queryを'from:ub_pnr -unformedbuilding.com'などのようにします。

検索コマンドについては下記を参照してください。
