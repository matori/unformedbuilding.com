以前、「WordPressの管理画面を自分の好きなフォントにするプラグイン」という記事を書いたんですが、これを少し変更してユーザーの権限ごとに違うスタイルを適用させることもできるバージョンを作りました。
やっていることは前と同じで、管理画面にスタイルシートを追加しているだけです。

いちいち作るのも面倒でしょうから、今回はダウンロードできるようにしました。
名前は「WP Custom Admin CSS」ということで。
対応バージョンはWordPress 2.8以降です。

「WP Custom Admin CSS」ダウンロード

使い方ですが、ダウンロードしたファイルを解凍して、解凍してできたフォルダ内のCSSファイルを編集して、管理画面に自分好みのスタイルを適用します。

roleフォルダ内にはユーザー権限ごとに適用できるCSSファイルが入っていますので、必要ならばそちらも編集してください。
権限別スタイルシートを利用する場合はwp-custom-admin-css.phpの35行目と65行目を削除して、コメントアウトを解除してください。

あとはwp-content/pluginsにアップロードすれば大丈夫だと思います。

機能の追加とかは自分が必要になったらします。
スタイルシートの書き方が分からない人は、勉強するか調べるかしてください。
